Direct 3D Frameork

My first attempt at laying the foundations for creating game engines

Working Direct with Direct 3D

One of the key APIs to develop graphical applications is Direct 3D. In this university hand in we were asked to create a scene using Direct 3D 11. This is my first time using Direct 3D, but not my first time using 3D graphic APIs. I have had previous experience using Open GL 1.4 and Vulkan to a very limited extent. But this was the first time been given so much control over the scene I was creating.

Furnishing the framework

The main learning experience for me was experimenting with different ideas when it came to structure my framework. One idea I toyed with was a flotilla. The idea of similar objects from a same class being grouped together which is a simplistic idea but helped when it came to the simplicity of rendering multiple objects and managing them, which is the goal of a framework. To make working with complex APIs or code simpler. Another learning experience was having all the ideas of working with Open GL reinforced.
