Open GL Enviroment

My first foray into Graphics APIs

Open Space

This project was made in accordance to my final assignment in my first year where we were asked to create an open environment with objects, that are read from a file and manipulatable. All of this was to be made with OpenGL 1.4. For me this was a brand new experience as it was my first time using an industry standard API (even if it was from 2 decades ago, and no I don’t count the Vulkan experiment in Pac man) and it presented new challenges such as fully setting up graphics API’s and reading objects from a file.

Only the Beginning of an Adventure

With the solution I provided I gained lots of experience when it comes to working with these sorts of API’s. While OpenGL 1.4 handles a lot of the more complex aspects of the graphics pipeline due to its fixed nature, it still provides a taster of what’s to come. In my solution there are multiple cubes and pyramids. These objects move on their own and there are multiple ways of manipulating and changing the object in the scene. After this experience with OpenGL I am looking forward to working with other API’s such as Direct3D.
