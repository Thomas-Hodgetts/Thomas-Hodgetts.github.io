Pacman Using C++ and S2D

My first adventure into programming games in C++

The Vulkan Venture

I first started trying to build my Pacman I tried to replace S2D with Vulkan. I wanted to do this as I wanted to gain experience in Vulkan. So using the Vulkan tutorials provided by the Kronos group and tried to build my game around that. This was too ambitious and I quickly realised that I was not going to be able to complete the project on time before the deadline was due.

While I had made progress with setting up and instance, and then rendered an 2D object in a window, I was not progressing nearly fast enough to meet my deadlines, so I made the choice to switch from Vulkan back to S2D midway through my assignment
